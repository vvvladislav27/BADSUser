# Фронтенд приложения по продаже БАДов (Telegram Mini App)

Этот репозиторий содержит проект фронтенда для приложения пользователя, написанного на Vue.js.

## Описание проекта

Данный проект предоставляет пользователям удобный интерфейс для поиска и покупки БАДов. Приложение позволяет пользователям:

- Просматривать каталог доступных БАДов
- Получать информацию о каждом продукте
- Оформлять заказы прямо в мессенджере
